## 关于

搭建一套自动化的项目创建工具，目标是管控项目整个声明周期

## 起步

- 安装：

```bash
$ npm install -g @winfe/winex-cli
# OR
$ yarn global add @winfe/winex-cli
```

- 创建一个项目：

```bash
$ winex init  -d my-project
```
